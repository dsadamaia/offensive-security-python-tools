## Installation

Clone the repository:

```bash
git clone https://github.com/dsadamaia/offensive-security-python-tools.git
```

Move into the project directory:

```bash
cd offensive-security-python-tools
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## Usage

Example:

```bash
python tool.py --help
```

or

```bash
python recon.py -d example.com
```
